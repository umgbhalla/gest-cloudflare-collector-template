# Self-contained Gest Cloudflare collector template

Generated by pnpm run build:cloudflare-template-release.

This directory vendors the built @gest runtime package closure under vendor/
and rewrites the template dependencies to local file: dependencies. Publish
this generated directory to a public GitHub/GitLab repository if you want a
Deploy to Cloudflare button before the @gest packages are published to npm.

1. @gest/ingest-core
2. @gest/client
3. @gest/ingest-cloudflare
4. @gest/ingest-discord
5. @gest/ingest-dispatch
